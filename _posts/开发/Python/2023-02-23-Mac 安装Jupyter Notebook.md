Mac 安装Jupyter Notebook

安装:

```javascript
pip install --upgrade pip
pip install jupyter
mkdir workspace/jupyter-test

```



运行：

```javascript
cd workspace/jupyter-test
jupyter notebook

```



可以在浏览器8888端口访问notebook
