在cmd中通过【tasklist】指令来查看进程 可用 指令【tasklist /?】来查看指令帮助 在cmd中通过【taskkill】指令来结束进程，例如【taskkill /im python.exe /f】来结束所有的python进程

```
taskkill /pid {pid} /f

```
